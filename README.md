# Marketing Attribution Analysis

## Description
This project performs Marketing Attribution Analysis using SQL and Excel.

Multi-channel marketing touchpoint data was cleaned and transformed using SQL. 
Customer journeys were created and attribution models such as First-Touch, 
Last-Touch, and Linear Attribution were applied to evaluate channel contribution.

Channel ROI was calculated by combining attribution revenue with marketing costs, 
and results were summarized using Excel with charts and insights.

## Technologies Used
- SQL (MySQL / MS SQL compatible)
- Microsoft Excel

## Attribution Models
- First Touch Attribution
- Last Touch Attribution
- Linear Attribution

## Deliverables
- Cleaned data in SQL tables
- SQL queries for attribution and ROI
- Excel summary report with charts and insights
